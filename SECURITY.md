# Security Policy

## Supported versions

Only the latest published version of `orbcode` on npm receives security fixes.

## Reporting a vulnerability

Please **do not** report security vulnerabilities through public GitHub
issues.

Instead, report them privately via
[GitHub Security Advisories](https://github.com/MatterAIOrg/OrbCode/security/advisories/new).

Include as much of the following as you can:

- the version affected (`orbcode --version`)
- steps to reproduce or a proof of concept
- the impact you believe it has

We will acknowledge your report as quickly as we can, keep you informed of
progress, and credit you in the fix release unless you prefer otherwise.
